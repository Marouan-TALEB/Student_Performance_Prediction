This project is a Streamlit-based web application that predicts student performance scores based on study habits and lifestyle factors.

## Prerequisites

- **Python 3.9 or higher**
- **pip** (Python package installer)

## Installation

### 1. Clone or Download the Project
Download the project files to your local machine.

### 2. Set Up a Virtual Environment (Recommended)

#### **Windows**
```bash
python -m venv venv
.\venv\Scripts\activate
```

#### **Linux**
```bash
python -m venv venv
source venv/bin/activate
```

### 3. Install Dependencies
Install the required Python packages using the `requirements.txt` file:
```bash
pip install -r requirements.txt
```

## Running the Application

To start the Streamlit application, run the following command in your terminal:

```bash
streamlit run app.py
```

Once the command is executed, the application will be accessible at `http://localhost:8501`.

## Project Structure

- `app.py`: The main Streamlit application file.
- `model.pkl`: The trained machine learning model.
- `requirements.txt`: List of required Python packages.
